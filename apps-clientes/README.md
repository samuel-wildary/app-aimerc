# Aplicativos dos clientes

Cada supermercado possui um projeto Android independente nesta pasta. Isso permite publicar cada marca separadamente na Play Store, com pacote, icone, cores e configuracao proprios.

## Estrutura

- `mercadinho-queiroz`: aplicativo do Mercadinho Queiroz.

Ao criar um novo aplicativo, use `android-customer-app` como base, defina um `applicationId` exclusivo e conecte o `STORE_SLUG` ao cadastro correto no SaaS.
